# 🌐 Personal Portfolio Website

This is a responsive personal portfolio website built using HTML, CSS, and JavaScript. It showcases my skills, projects, education, and contact details in a modern UI design.

---

## 🚀 Features

- 📱 Fully Responsive Design (Mobile + Desktop)
- ☰ Mobile Menu (Hamburger Navigation)
- 👨‍💻 About Me Section
- 🛠 Skills Section (Technical & Professional)
- 📂 Projects Showcase
- 🎓 Education Timeline
- 📄 Resume View & Download
- 📞 Contact Form Section
- 🔗 Social Media Links

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
- Boxicons (Icons)
- Typed.js (Text animation)
